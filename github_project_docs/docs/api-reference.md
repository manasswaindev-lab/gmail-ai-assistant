# API Reference

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/v1/login | Start OAuth |
| GET | /api/v1/auth/callback | OAuth callback |
| GET | /api/v1/gmail/messages | List emails |
| GET | /api/v1/gmail/messages/{id} | Read email |
| GET | /api/v1/ai/analyze/{id} | Analyze email |
| POST | /api/v1/ai/reply/{id} | Generate AI reply |
